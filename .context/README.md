# .context/

This directory is a machine-readable context bundle for **lumiko**,
generated by [Lumiko](https://github.com/mirako-dev/lumiko).

It's designed to be loaded by AI coding agents (Claude Code, Cursor, Copilot,
custom RAG pipelines) so they can understand this codebase without having to
re-read every source file.

## Layout

- `manifest.json` — index of all files in this bundle
- `overview.json` — project summary, stack, entry points
- `architecture.json` — system design, modules, data flow
- `conventions.md` — coding conventions
- `commands.json` — build, test, dev commands + env vars
- `modules/` — one file per directory with per-file details

## Regenerating

Run `lumiko generate` from the project root to refresh this bundle.
